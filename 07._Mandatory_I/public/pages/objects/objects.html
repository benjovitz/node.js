Objects

Javascript objects are initiated with curly brackets { }.
They are used to store multiple values in a single instance in an organized way. They also align with JSON syntax which makes them very easy to work with.

Below are some examples for altering an object.

Access a single property of the object with objectName.propertyName.
Add new values with objectName.newPropertyName.
All types are welcome in an js object, may they be strings, numbers, booleans, functions or arrays. Even other objects.

The specific properties can be accessed with this which refers to itself. So in the function calling this.name return the name within the object. This helps with scoping so we dont have to use variable names from potentially other scopes. 

        // js objects
        const person = {
            name: "Daniel",
            age: 22,
            height: 182,
            learningLevel: "Chimp",
            makeNoise: () => "*SCREAMING VIOLENTLY*",
            get scream () {
                return `*SCREAMING* "MY NAME IS ${this.name.toUpperCase()}`
            },
            numbersIKnow: [1, 2, 3],
            pet: {type: "Cat", name: "Bitten", age: 1},
            winning: true
        }
        
        person.age = 23 
        console.log(person)
        
        console.log("learning level =", person.learningLevel)
        
        delete person.age
        console.log(person)
        
        person.drink = "Apple juice"
        console.log(person)

        console.log(person.makeNoise())
        console.log(person.scream)

⚠️ Objects Beware! ⚠️

Types like number, strings and booleans are primitives. When you change the value of a primitive a new value is actually created instead of chaning the old one in the memory.

Objects however are complex data types. When you change an object you change the specific value in that memory address.
That also means that if you let one object be equal to another, changes will happens to both, since they are both referencing the same address in the memory. 
        let primitive = 7
        primitive = 10 // new memory address

        const monkey1 = {name: "Daniel", age: 23, type: "Human"}
        const monkey2 = {name: "Hiro", age: 15, type: "Chimp"}

        const monkey3 = monkey1 // same address reference 

        monkey3.age = 6

        console.log("Monkey 1:", monkey1)

Date Object

Getting todays date can be very useful and javascript has a couple of built in function for just that!
Date() returns todays date as a string.
new Date() returns a date object, which has getters on properties milliseconds, seconds, minutes, hours, day, month and year. The date object also has function called toLocalDateString which takes two arguments (it also sets the timezone to the users timezone). First is the language and format of the string like "da-DK" for danish. The second argument is an object of options for the string like which values of the time to return such as only the month, day and year, and have the year only displayed with 2 digits. A timezone can also be specified.
Date.now() returns the number of milliseconds elapsed since midnight, January 1, 1970. The date was chosen by the engineers of the UNIX system which started development in 1969.
