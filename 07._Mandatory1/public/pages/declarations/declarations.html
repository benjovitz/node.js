const and let is only available in their given scope, seen in our block scope (line 7-10)
So when we try to print the not global variable we get an error, since the log statement and notGlobal variable is in different scopes.
let can be reassigned and const cant (line 2-5)
var is available in the whole js file or global scope, var can be reassigned. Thats why it is available for our console.log outside the block scope
totalGlobal is availble over the whole project, it can be reassigned (line 15)
const can still be altered, in line 17 we have an array which is a const, but it can still take in new values in the array.
