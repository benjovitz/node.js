Git in the Terminal

Working with git is essential for a developer for accessing and sharing code. Working with it in the terminal is the rawest form wich doesnt make you dependet of a program or extension to push or pull your code.

Clone an existing repository

Pushing to a repository

We now have a repository with where to push is already defined in its .git folder. So pushing to it is easy!
git add "filename" to add a specific file to your commit or git add -A to add all changes. 
 git add -A
