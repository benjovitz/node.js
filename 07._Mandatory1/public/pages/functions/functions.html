Functions

Functions allows write logic down once, instead of repeating ourselves. This makes the code more maintainable, reuseable and modular.
Hoisting means that our functions will be read first, wich allows us to call it before initialization. Const and let are hoisted but not initialized. They need to be declared before usage. Var is hoisted and initialized but bleeds to other scopes, so could be a bit tricky to use.
Callback is when a function is taken as a parameter in another function.
The return is the last thing happening in the function, code after the return statement wont be executed. the return statement is that function delivers back to you. Heres a couple of ways to write functions in javascript.
