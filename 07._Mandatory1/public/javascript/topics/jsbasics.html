Javascript

Javascript is a language used to run code in the browser. To run javascript in our terminal we need to download node.js

What is node.js?

Node.js is a tool that makes it possible to run javascript outside the browser. The engine is taken from chrome in 2009. npm (node package manager) was introduced later with the purpose of installing module to your project.

Getting node and run javascript files

Starting out by donwloading node.js
Lets create a js file called "app.js" and make a console.log("Hello World!") to see the output in the terminal. with node we are able to run js files locally instead of the browser with this command 
app.js
            
            console.log("Hello World!")

Strings

Strings can be concatenated with + as seen in the first console.log( ).
If the string represents a number it can also be parsed as such with Number, seen in the second console.log( ).
Strings are in fact an array of chars, so we can access every part of the string as an array, seen in third console.log( ).
We can also replace values in strings, seen i fourth console.log( ).
