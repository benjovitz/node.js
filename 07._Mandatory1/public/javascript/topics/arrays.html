Arrays

(some) Array methods explained

Theres a bunch of array methods built into Javascript here are some i have used.
Specific values in the array can be accessed with array[index].
Get the size of the array with array.length which returns an integer of the arrays length.
Join the whole array together with array.join("something between each element") returns a string with the chosen separator between each element.
Remove the last element of the array with array.pop() which returns the element that was removed.
Add an element to the end of the array with array.push("new element") which returns an the new size of the arrray as type number.
Remove the first element of the array with array.shift() and returns the element that was removed, all elements are shifted down 1 index.
Add an element to the start of the array with and shifts all elements one index up array.unshift("new element") which returns an the new size of the array as type number.
Take a slice of the array with array.slice(startIndex, optionalEndIndex) which returns a new array consisting of elements from the array that was sliced, from start index to end index or if no end index was set, from start index to the end of the array. This doesnt affect the old array.
With array.splice(startIndex, optinalDeleteCount, "item1", ..."itemX") you can both remove and add elements to the array. The method returns an array of the removed elements. Start out by giving a startIndex, then an optional number of items to delete (0 or not given also an option). Then optional, elements to add from the start index and shift other elements that would come after the start index. 
        const arr = ["Red", "Green", "Blue", "Black", "White"]

        console.log(`slice method: ${arr.slice(2, 5)}`)
        console.log(arr)

        console.log(`splice method: ${arr.splice(3, 1, "monkey business")}`)
        console.log(arr)
