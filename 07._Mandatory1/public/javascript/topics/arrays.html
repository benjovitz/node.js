Arrays

(some) Array methods explained

Theres a bunch of array methods built into Javascript here are some i have used.

array[index]

How to use: array[number]
What it does: gets element with given index
Returns: a single element from the array || undefined

array.length

How to use: array.length
What it does: gives length of array
Returns: number

array.join

How to use: array.join(string)
What it does: joins the array in a single string with chosen separator between each element
Returns: string

array.pop

How to use: array.pop()
What it does: removes the last element of the array
Returns: the removed element

array.push

How to use: array.push(element)
What it does: appends element to the end of the array
Returns: size of array type number

array.shift

How to use: array.shift()
What it does: removes first element of the array and moves all other elements an index down
Returns: the removed element

array.slice

How to use: array.slice(startIndex, endIndex)
What it does: creates new array from startIndex to endIndex or if not given, to the end of the array
Returns: array

array.splice

How to use: array.slice(startIndex, deleteCount, element)
What it does: can both add and remove items from array. Choose startIndex from where to start count, deletecount is how many items is deleted, can be not given. Elements to add from start index and shift everything after +1 index
Returns: array of removed items

Examples

        const arr = ["Red", "Green", "Blue", "Black", "White"]

        console.log(arr) 

        console.log(arr[2]) 

        console.log(`length: ${arr.length}`)

        console.log(`join method: ${arr.join(" and ")}`)

        console.log(`pop method: ${arr.pop()}`)
        console.log(arr)

        console.log(`push method: ${arr.push("Pink")}`) 
        console.log(arr)

        console.log(`shift method: ${arr.shift()}`)
        console.log(arr)

        console.log(`unshift method: ${arr.unshift("Purple")}`)
        console.log(arr)

        console.log(`slice method: ${arr.slice(2, 5)}`)
        console.log(arr)

        console.log(`splice method: ${arr.splice(3, 1, "monkey business")}`)
        console.log(arr)
