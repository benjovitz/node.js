Javascript

Javascript is a language used to run code in the browser. To run javascript in our terminal we need to download node.js

What is node.js?

Node.js is a tool that makes it possible to run javascript outside the browser. The engine is taken from chrome in 2009. npm (package manager) was introduced later with the purpose of installing module to your project.

Gettng node and run js files

Starting out by donwloading node.js
Lets create a js file called "app.js" and make a console.log("Hello World!") to see the output in the terminal. with node we are able to run js files locally instead of the browser with this command
